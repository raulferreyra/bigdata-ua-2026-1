## ¿Qué aprendí?

En este laboratorio aprendí a diferenciar de manera práctica los datos estructurados, semi-estructurados y no estructurados dentro del ecosistema de Big Data. También comprendí cómo simular miles de registros comerciales en Python para analizar patrones de consumo regional y métodos de pago como Yape y Plin. Finalmente, aprendí a estructurar una propuesta de arquitectura en la nube usando herramientas clave como Apache Kafka y Amazon S3.

## ¿Qué fue lo más difícil?

Lo más difícil fue comprender inicialmente la sintaxis de Git para sincronizar las ramas del repositorio sin perder los archivos locales, y estructurar correctamente el diccionario de arquitectura Big Data en Python respetando el formato JSON.

## ¿Cómo aplica en mi empresa actual?

Aplica directamente ayudando a automatizar la ingesta y el análisis de grandes volúmenes de transacciones diarias que se generan en tiempo real. Esto nos permitiría detectar fraudes al instante y predecir la escasez de stock en los almacenes antes de que afecte a las tiendas físicas.
